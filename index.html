Technologien Damit arbeiten wir gerne

Um eine Internet Plattform oder eine Software Lösung zu entwickeln, setzen wir verschiedene Technologien ein. Wir wählen unter anderem anhand folgender Kriterien die Technologien aus:

  • Kosteneffizienz: Wieviel kostet die Entwicklung?
  • Geschwindigkeit: Wie schnell kann die gewünschte Lösung entwickelt werden?
  • Stabilität: Wie kann eine genügend hohe Stabilität erreicht werden?
  • Wartbarkeit: Wie teuer ist die Software im Unterhalt?
  • Agilität: Wie einfach können Änderungen durchgeführt werden?
  • Verbreitung und Community: Wie verbreitet ist die Technologie und wie schnell entwickelt sich diese weiter?

Web
HTML5, CSS3 und Javascript

Wir arbeiten mit modernsten Web Technologien, weil heute der grösste Teil der Bevölkerung Internetfähige Geräte (PC, Laptop, Handy, ...) besitzt, mit denen das Internet genutzt werden kann. Zudem entwickeln sich diese Technologien sehr schnell, wodurch immer wieder neue Möglichkeiten für neue Produkte und Lösungen bietet und Raum für Innovation lässt.

Mit HTML5, CSS3 und Javascript ist es für einen Nutzer möglich, ohne eine zusätzliche Softwareinstallation direkt auf die Services zuzugreifen.

Diese Technologien sind auch am meisten verbreitet, wodurch sehr viele Endgeräte angesprochen werden können, vom IPhone über ein Android Tablet bis zum Desktop PC zuhause oder im Büro.

Ruby on Rails
Schnelle Entwicklung

Um der Applikation ein kräftiges Backend zu verleihen, das schnell, einfach und kosteneffizient entwickelt werden kann, setzen wir auf Ruby on Rails.

Durch Konvention über Konfiguration ist es den Entwicklern möglich, dank logischen Standardwerten nur die Werte anzupassen, die für die spezifische Lösung benötigt werden. Dadurch wird weniger Code produziert, wodurch die Entwicklung beschleunigt und die Wartung effizienter wird.

Ruby on Rails wurde schon in vielen Projekten eingesetzt (zum Beispiel bei Twitter, Groupon, Xing, usw.) und überzeugt immer wieder mit Innovation. Weitere Informationen zu Ruby on Rails finden Sie unter: http://rubyonrails.org

jQuery
DIE Javascript Bibliothek

Während früher Internetseiten meistens statisch und ohne Javascript funktionierten, ist es heute nicht mehr wegzudenken, ohne Javascript zu arbeiten. Da viele verschiedene Endgeräte und Browser unterstützt werden müssen, setzen wir jQuery ein.

jQuery vereinfacht die Bearbeitung von HTML direkt auf dem Endgerät. Dadurch ist es z.B. möglich, Animationen oder Dialoge einzublenden. Weiter ermöglicht jQuery sogenannte AJAX Abfragen, damit nicht immer die ganze Seite neu geladen werden muss, sondern nur ein kleiner Teil bearbeitet wird.

Die Verbreitung von jQuery im Web spricht für sich (je nach Quelle bei 50% oder mehr Internetseiten verwendet). Die aktive Community entwickelt diese Bibliothek auch stetig weiter, damit neue Browser und neue Standards stehts unterstützt werden.
