Linguaservice Implementierung und grafische Aufbereitung effizienter Suche

Ausgangslage

LinguaService bietet Dienstleistungen für Sprachreisende auf der ganzen Welt. Aus diesem Grund musste eine Webseite her, in welcher die einfache und effiziente Auffindbarkeit einer passenden Destination bzw. Sprachschule im Mittelpunkt steht.

Produkt

Verschiedene Filterkriterien helfen dem Nutzer beim Finden der passenden Schule. Zudem werden die Suchresultate grafisch aufbereitet und auf einer Landkarte angezeigt. Eine übersichtliche Menustruktur erhöht die Usability weiter.
